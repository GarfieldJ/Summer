# Live2D Widget

## 目录结构 Files

- `waifu-tips.json` 中包含了触发条件（`selector`，选择器）和触发时显示的文字（`text`）；
- `waifu.css` 是看板娘的样式表。

可以在官方网站下载、打包 SDK，以获取更多的功能。具体操作是：
- 点击Cubism SDK for Web，下载相关文件；
- 解压并进入目录，执行 `npm install`；
- 执行 `npm run build-sample`。
